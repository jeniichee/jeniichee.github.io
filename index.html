battlesaslvo

BattleSalvo

A Java command-line rendition of the classic game Battleship with a twist. Features flexible board sizes, set ship sizes, and a unique shooting order. Supports both single-player mode against an AI opponent and multiplayer mode via sockets.

maze

Maze

Players navigate mazes, randomly generated using Kruskal's algorithm, from the top-left corner to the bottom-right corner. Different maze dimensions are supported, and two solving algorithms are offered: breadth-first search (BFS) and depth-first search (DFS). The maze is rendered using a graphical interface, with start and end cells highlighted, visited cells marked, and the solution path displayed.

second-chance

Second-Chance Co

An online marketplace for secondhand clothing, offering a platform for individuals to sell their clothes to others. Users can upload photos and descriptions of their items, browse products, add items to their cart, and manage their profile. The project utilizes a MySQL 8 container, a Python Flask container for the REST API, and a local AppSmith server for deployment.

bullet journal

Bullet Journal

An interactive application developed using SceneBuilder, JavaFX, and Java to track events and tasks for individual weeks. Features included: week view, event and task creation, persistence using JSON encoding, commitment warnings for maximum events and tasks, task sidebar, categorization of events and tasks, quotes and notes section, deletion of tasks and events, weekly overview statistics, editing existing events and tasks, and auto tags for categories. The goal is to create a customizable and user-friendly application that simulates the experience of a physical bullet journal.
