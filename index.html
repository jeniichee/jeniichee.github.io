Projects

BattleSalvo

Java command-line rendition of classic Battleship with a twist. Supports both single-player vs. AI and multiplayer via sockets. Communication between server and client are serialized into files to ensure persistance.

maze

Players navigate mazes from the top-left corner to the bottom-right corner, and solves it using either BFS or DFS. The mazes are randomly generated using Kruskal's Algorithm. Graphical interface renders cells and highlights actions.

second-chance

Platform for individuals to sell their clothes to others. Users can upload photos and descriptions of their items, browse products, add items to their cart, and manage their profile. The project utilizes a MySQL 8 container, a Python Flask container for REST API, and a local AppSmith server for deployment.

bullet journal

Interactive application developed with SceneBuilder, JavaFX, and Java to track events/tasks for individual weeks. Features: week view, event/task creation, persistence using JSON encoding, deletion of events/tasks, editing existing events/tasks, and categories.
